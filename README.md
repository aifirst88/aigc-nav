# AI工具导航站

一个现代化的AI工具导航网站，类似 aigc.cn。

## 技术栈
- Next.js 14
- React 18
- Tailwind CSS
- TypeScript

## 部署

### Vercel 一键部署

1. 将代码推送到 GitHub
2. 在 [Vercel](https://vercel.com) 导入项目
3. 自动部署完成！

### 本地开发

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 功能
- 🔍 工具搜索
- 🏷️ 分类筛选
- 🔥 热门推荐
- 📱 响应式设计
- 🌙 暗黑模式
